# 🍇 Grapes

**Dalla cantina alla tavola** — il marketplace della vendita diretta delle cantine italiane. / **From the cellar door to the table** — the direct-sale marketplace for Italian wineries.

Il consumatore legge la storia vera di ogni cantina in un minuto e compra i suoi vini al prezzo di cantina, senza i 3–4 passaggi della distribuzione classica. Prototipo dimostrativo bilingue IT/EN (ordini simulati, nessun pagamento, 18+).
